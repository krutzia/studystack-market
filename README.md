# CampusKart

CampusKart is a React, Vite, and Tailwind CSS app for buying and selling books, notes, gadgets, and other student essentials within a campus community.

## Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Testing

```bash
npm run test
```
